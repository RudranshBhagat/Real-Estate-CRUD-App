# Real Estate CRUD App

## 📌 Overview

This is a full-stack Real Estate Property Listing application that allows users to perform CRUD (Create, Read, Update, Delete) operations on property data.

---

## 🚀 Features

* Add property listings
* Edit property details
* Delete properties
* View all properties

## ⚙️ Setup Instructions

### 1. Clone the repository

```
git clone https://github.com/RudranshBhagat/Real-Estate-CRUD-App.git
```

### 2. Setup Database

* Open SQL Server Management Studio (SSMS)
* Run SQL file from `/database/property_listings.sql`

### 3. Run Backend

```
cd backend
npm install   # or your backend setup
npm start
```

### 4. Run Frontend

```
cd frontend
npm install
npm start
```
